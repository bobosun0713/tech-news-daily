## 深入閱讀

### 1. [Hugging Face 與 Cerebras 推進 Gemma 4 即時語音 AI (Hugging Face and Cerebras bring Gemma 4 to real-time voice AI)](https://huggingface.co/blog/cerebras-gemma4-voice-ai)

### 2. [ScarfBench：評測 AI 代理遷移企業 Java 框架的能力 (ScarfBench: Benchmarking AI Agents for Enterprise Java Framework Migration)](https://huggingface.co/blog/ibm-research/scarfbench)
* **核心重點：** 用基準測試檢驗 AI 代理在企業框架遷移任務中的可用性，而不只看聊天能力。
* **相關標籤：** `#Benchmarking` `#Agents` `#EnterpriseJava`

### 3. [Hugging Face 模型頁面直接展示每個 Eval 結果 (Featuring Every Eval Ever Results on Hugging Face Model Pages)](https://huggingface.co/blog/eee-community-evals)
* **核心重點：** 把評測結果前置到模型頁面，讓比較、篩選與模型挑選更透明。
* **相關標籤：** `#Evals` `#ModelCards` `#OpenSource`

### 4. [BaseRT：支援 Apple Silicon 的本地 AI 高速推理執行環境 (BaseRT, A fast inference runtime for local AI on Apple Silicon)](https://www.basecompute.co/getbasert)
* **核心重點：** 針對 Apple Silicon 優化本地推理執行，對離線與低延遲 AI 應用特別實用。
* **相關標籤：** `#InferenceRuntime` `#AppleSilicon` `#LocalAI`

### 5. [aiCompiler：用 Markdown 寫意圖，讓 LLM 以 runtime 執行 (Crazy idea? aiCompiler – write intent in Markdown, LLM executes it as a runtime)](https://aicompiler.dev)
* **核心重點：** 嘗試把語意指令變成可執行 runtime，延伸了 LLM 不只產生文字，也能驅動流程的想像。
* **相關標籤：** `#LLM` `#Runtime` `#Workflow`
